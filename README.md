Ohjelma, joka antaa käyttäjälle matematiikan, fysiikan ja kemian laskuja ja kysymyksiä. Kysymysten muuttujat arvotaan aina kun ladataan uusi kysymys, jolloin pienempikin määrä valmiita tehtäviä tuottaa monipuolisen määrän laskuja joihin vastata. Ohjelma laskee käyttäjän oikein vastattujen tehtävien määrän, ja kykenee antamaan käyttäjälle vastauksen tai mallivastauksen tehtävään sitä pyydettäessä.
